# 🖥️ Central de chamadas — Sistema de Atendimento e Chamadas

Sistema web desenvolvido em **PHP + CodeIgniter**, focado na gestão de atendimento, controle de senhas e organização de fluxo em ambientes operacionais.

O projeto simula um cenário real de uso, com regras de negócio aplicadas para controle de chamadas, pacientes e exibição em painel.

---

## 🚀 Visão geral

O **Central de chamadas** é um sistema voltado para ambientes que trabalham com fluxo de atendimento, como:

- Clínicas
- Recepções
- Ambientes com controle de fila
- Sistemas de chamada por senha

A aplicação permite organizar, chamar e exibir atendimentos de forma clara e funcional.

---

## 🧠 Principais funcionalidades

- 📢 Geração e chamada de senhas
- 👥 Controle de pacientes / atendimentos
- 📺 Exibição em painel (display de chamadas)
- 🔄 Atualização dinâmica via endpoints JSON
- ⚙️ Organização de fluxo operacional
- 🧩 Estrutura MVC com CodeIgniter

---

## 🛠️ Tecnologias utilizadas

- PHP
- CodeIgniter 3
- JavaScript
- HTML5 + CSS3
- Arquitetura MVC
